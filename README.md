# AttendX AI

AttendX AI is a Python-based attendance system that uses computer vision to automate attendance recording.

## Features
- Face detection using OpenCV
- Automatic attendance logging
- Simple and efficient system for classrooms

## Technologies Used
- Python
- OpenCV
- NumPy

## How to Run

1. Install required libraries:
pip install -r requirements.txt

2. Run the program:
python main.py
